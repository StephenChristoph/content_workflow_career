Week 1: Writing Cycle

Objective

Create foundational long-form content that will serve as the basis for all other content formats.

Key Activities

  • Content Planning: Review trending topics, audience feedback, and content calendar
  • Writing: Create 2-3 long-form pieces (1500+ words each)
  • Platform Optimization: Adapt content for Substack, Medium, and Typeshare specific requirements

Step-by-Step Process

  1. Open Notion and review content calendar
  2. Draft outline for each piece
  3. Use AI writing tools to assist with research and initial drafts
  4. Edit and polish content
  5. Prepare platform-specific versions
  6. Schedule publication dates

Best Practices

  • Focus on evergreen topics that can be repurposed
  • Include clear takeaways and actionable insights
  • Optimize headlines for each platform
  • Include relevant keywords for SEO

Week 2: Iteration Cycle

Objective

Transform long-form content into engaging short-form posts for social media platforms.

Key Activities

  • Content Breakdown: Extract key points from Week 1's content
  • Platform Adaptation: Create platform-specific versions
  • Schedule Management: Plan posting timeline

Step-by-Step Process

  1. Review Week 1's content and identify key takeaways
  2. Create content calendar for social posts
  3. Adapt content for each platform:
    • LinkedIn: Professional insights
    • X: Concise threads
    • BlueSky: Conversation starters
    • Threads: Engaging narratives
  4. Set up scheduling in social media management tools
